#Customer Segmentation Project

## Overview

This project uses K-Means Clustering to segment customers based on their Annual Income and Spending Score.

## Objective

To identify different customer groups and understand customer purchasing behavior.

## Features

- Data Analysis
- Customer Visualization
- Elbow Method
- K-Means Clustering
- Customer Segmentation
- Business Insights

## Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-Learn
- Google Colab

## Results

Customers were grouped into different clusters based on income and spending patterns. These insights can help businesses create targeted marketing strategies.

## Future Scope

- Larger datasets
- Real-time customer analytics
- Interactive dashboards
